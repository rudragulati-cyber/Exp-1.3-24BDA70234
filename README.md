# Exp-1.3-24BDA70234
Aim

To design and develop an Admin Dashboard using HTML, CSS Grid, and JavaScript with dynamic theme switching.

Software Requirements

Web Browser (Chrome / Firefox / Edge)

Text Editor (VS Code / Notepad++)

Technologies Used

HTML

CSS (Grid Layout)

JavaScript

Description

This experiment demonstrates the creation of an admin dashboard interface using CSS Grid for layout design.
The dashboard includes a sidebar, header, and multiple information boxes.
A theme toggle feature is implemented using JavaScript to switch between light and dark modes.

Learning Outcome

After completing this experiment, the learner will be able to:

Understand the basic structure of an admin dashboard

Apply CSS Grid to create responsive layouts

Design and style dashboard boxes using CSS

Use CSS variables for theme management

Implement light and dark theme switching using JavaScript
